Narrrowing - Verificação de tipos e dados com Typescript

Narrowing é um recurso de Typescript para identificar tipos de dados, dando assim uma direção diferente à execução do programa, baseada no tipo que foi identificado.

Há situações em que os tipos podem ser imprevisíveis (exemplo, dados vindos de uma API) e queremos executar algo para cada uma das possibilidades. Isso é fundamental para evitar erros do compilador.
